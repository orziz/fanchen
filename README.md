# 凡尘立道录

本项目是一个本地运行的修真经营与挂机游戏，核心循环围绕修炼、跑商、置业、势力经营与宗门成长展开。

## 项目硬规则

### 游戏内文案规则

- 游戏内禁止输出任何制作过程、开发讨论、版本自述、布局解释、AI 来源说明、临时占位文案、调试信息或其他干扰正常游玩的无用信息。
- 游戏内文本只能服务于以下几类内容：世界观与氛围、事件与结果反馈、玩法状态、必要且最短的操作提示。
- 新增按钮文案、日志、面板说明、提示语时，优先写成世界内表达；如果不适合写成世界内表达，就只保留最短必要信息。
- 发现已有文案带有“这一页现在如何调整”“某功能不再怎样”“界面怎么改过”“这是开发中的说明”这类表达时，应直接删除或改写。

### 修改约束

- 新功能必须优先接入现有世界、产业、势力、宗门与地图系统，不做悬空的演示式功能。
- 涉及挂机收益的改动，优先控制滚雪球速度，再给成长型投入与组织分工留空间。
- 涉及 UI 文案的改动，宁可更短，也不要把设计说明写进游戏本体。
