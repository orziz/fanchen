当前实现稿：

1. 玩家在青禾镇街口醒来。
2. 甲把人叫醒，玩家表明自己想拜入宗门，但盘缠已经见底。
3. 甲先让玩家认清青禾地界，不要急着空谈宗门。
4. 引导把山河图打开，并把剧情切到侧栏，方便玩家一边看图一边继续对话。
5. 甲发放第一包新手物资：木柄短枪、草膏、粗灵米、雾心草。
6. 引导把势力页点开，第一目标明确为“先在青禾挂靠一方势力”。
7. 玩家加入第一家势力后，商路、产业、群像、战斗、天机等其余核心面板与快捷操作再放开。

当前解锁节奏：

- 开场时只放开：行囊、调息、修炼、炼体。
- 看完地图节点后放开：山河图。
- 收下新手物资后放开：势力与宗门。
- 加入第一家势力后放开：其余核心面板与行商等后续门路。

原则不变：

- 功能要一步步解锁，不要一下全开给玩家。
- 第一个明确目标必须是能立刻闭环的生计与门路，而不是空悬的“拜入宗门”。
