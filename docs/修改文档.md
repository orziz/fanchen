# 当前进度

> 本文只记录当前实现进度与近期待做口径。
> 题材与世界规则以 [修真题材长期总纲](./修真题材长期总纲.md) 为准，系统先后顺序以 [修真系统分期总表](./修真系统分期总表.md) 为准。

## 已完成

- 主界面已稳定为左侧人物与案头、右侧主工作区的结构。
- 挂机策略与手动操作已迁入谋局台，不再与人物栏重复占位。
- 山河图与江湖纪事已改为独立浮窗。
- 浮窗已支持拖拽、关闭、最小化、基础停靠与布局记忆。
- 地图窗口已放大，地图与地脉详情的主次关系已重新分配。
- 产业系统已有田产、工坊、铺面与行会订单。
- NPC 已有年龄、阶段、职业、目标、势力流动与近年履历。
- 个人记录与阶段性统计已从常驻人物卡移入低频人物簿窗口。
- 案头里的重复“功能直达”已移除，避免和主工作区页签重复。
- 已加入“手动操作”模式，世界照常推进，但不会自动替玩家执行日常动作。
- 当前结算、面板展示与存档落盘已统一按整数口径处理；零碎增益改走内部进位，不再把小数直接留在角色与势力状态里。
- 自建宗门当前改为封存态：旧档宗门只保留账册、名录与概览查看，不再开放新建、收徒、传功、建筑与门内差使；外部势力与自家势力照常运转。

## 当前可用

- 主工作区：行囊、产业、商店、拍卖、战斗、群像、势力、天机。
- 谋局台控制：挂机策略切换、手动修炼、主动冲关、追查机缘、顺手做买卖、短暂调息。
- 低频窗口：山河图、江湖纪事、人物簿。
- 常驻高频信息：人物状态、当前焦点、最新风闻、活跃秘境、当前门路。
- 势力页中的宗门栏当前仅用于查看封存旧档；新的自建宗门玩法暂不开放。

## 接下来优先做

### 阶段一收口

- 条件不足与按钮不可用原因必须说清。
- 所有已开放功能必须真实有效，不能只开入口不给结果。
- 境界的作用、修为积累与突破火候的区别要彻底说明并落到玩法上。
- 酒馆、茶馆与消息打听要补上，承接地方风闻与见闻门槛。
- 存读档与基础中文文本继续收口。
- NPC 见过面或听说过才显示。

### 阶段二推进

- 家宅系统：买屋、扩院、仓房与工位摆放。
- 委派系统与任务时限。
- 势力、官府、宗门、个人与玩家都能发委派，NPC 能执行委派。
- 地区治安与税赋：直接影响跑商、铺面和产业收益。
- 黑市、镖局、赌场：补足低阶生计路线。
- 更完整的窗口停靠与多区布局。

### 阶段三埋种

- 师承谱系：NPC 收徒、带徒与传承链。
- 婚配与家族：道侣、子嗣、继承与家业流转。

## 还没做满

- 目前只有地图、纪事、人物簿进入了低频窗口体系，更多低频模块还没迁入。
- 窗口停靠还是基础版，尚未做吸附、多列分区和更完整的桌面编排。
- NPC 的婚育、家族、墓志与代际传承还未形成完整闭环，目前只适合当阶段三种子。
